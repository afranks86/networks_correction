Precision and recall in glasso networks using canonical pathway databases with GC bias as the known covariate

This will re-generate precision and recall plots in graphical lasso networks using canonical pathway databases and includes RIN, GC bias, exonic rate, Library size (smestlbs), rRNA rate (smrrnart), mean coefficient of variance (smmncv), and unique rate of mapped reads (smunmprt) as covariates
